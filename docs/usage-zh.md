# 使用指南

本指南介绍如何使用 ResearchClaw — 一个 AI 驱动的科研管理桌面应用。

---

## 概述

ResearchClaw 围绕四个核心工作流组织：

| 工作流   | 位置               | 功能                              |
| :------- | :----------------- | :-------------------------------- |
| **导入** | Dashboard          | 从 Chrome 历史批量导入论文        |
| **下载** | Dashboard / 论文页 | 通过 arXiv ID 或 URL 添加单篇论文 |
| **阅读** | 阅读器             | PDF + AI 生成笔记并排显示         |
| **创意** | 创意页面           | 从论文库生成研究方向              |

---

## 工作流 1：导入论文

### 从 Chrome 浏览历史导入

**步骤 1：在 Chrome 中浏览论文**

在 Chrome 中访问 arXiv.org 上的论文。应用会扫描本地 Chrome 历史文件。

**步骤 2：在 Dashboard 点击"从 Chrome 导入"**

应用会自动：

1. 解析 Chrome 历史中的 arXiv URL
2. 从 arXiv API 获取论文元数据
3. 将论文添加到本地数据库
4. 下载 PDF 以便离线阅读

Dashboard 上会实时显示导入进度。

### 下载单篇论文

在 Dashboard 或论文页面，点击**下载**按钮，输入 arXiv ID 或 URL：

```
2401.12345
https://arxiv.org/abs/2401.12345
```

---

## 工作流 2：AI 辅助阅读

### 打开阅读器

在**论文**页面，点击任意论文打开阅读器。

阅读器显示：

| 面板     | 内容                 |
| :------- | :------------------- |
| **左侧** | 富文本笔记（可编辑） |
| **右侧** | PDF 阅读器           |

### AI 笔记生成

点击 **"AI 填充"** 让 AI 分析 PDF 并生成结构化阅读笔记，涵盖：

- 问题描述
- 核心方法
- 关键结果
- 局限性
- 个人感悟

### 编辑模式

| 模式       | 行为                              |
| :--------- | :-------------------------------- |
| **Vibe**   | 查看 AI 生成的笔记（只读）        |
| **Manual** | 自由编辑笔记；`Ctrl/Cmd + S` 保存 |

点击 **Vibe / Manual** 切换按钮来切换模式。

---

## 工作流 3：生成研究创意

### 前提条件

先导入并阅读几篇论文，建立你的论文库。

### 生成创意

进入**创意**页面，点击**"生成创意"**。

AI 会分析你的论文库并产出：

- **趋势分析** — 你阅读中出现的新兴话题
- **研究方向** — 潜在的探索领域
- **新颖想法** — 现有工作的组合和扩展

每个创意包含标题、方向、假设、验证路径、优先级和新颖度评分。

---

## 工作流 4：配置 AI 提供商

进入**设置 → AI 提供商**，配置驱动应用的 AI 模型。

### 支持的提供商

| 提供商        | 模型                                                 |
| :------------ | :--------------------------------------------------- |
| Anthropic     | claude-opus-4-6, claude-sonnet-4-6, claude-haiku-4-5 |
| OpenAI        | gpt-4o, gpt-4-turbo, gpt-3.5-turbo                   |
| Google Gemini | gemini-2.0-flash, gemini-1.5-pro                     |
| 自定义        | 任意 OpenAI 兼容 API                                 |

### 设置步骤

1. 选择提供商
2. 输入 API Key
3. （可选）设置自定义 Base URL（用于代理或本地模型）
4. 点击**保存**
5. 设为**激活**，用于所有 AI 功能

---

## 工作流 5：CLI 工具

进入**设置 → CLI 工具**，直接在应用内运行 AI CLI 工具。

### 支持的工具

| 工具         | 命令     | 备注       |
| :----------- | :------- | :--------- |
| Claude Code  | `claude` | 需单独安装 |
| OpenAI Codex | `codex`  | 需单独安装 |
| Gemini CLI   | `gemini` | 需单独安装 |

应用会自动检测已安装的工具并显示版本号。使用终端面板运行命令，实时查看流式输出。

---

## 技巧与快捷键

| 操作         | 方法                         |
| :----------- | :--------------------------- |
| 保存笔记     | Manual 模式下 `Ctrl/Cmd + S` |
| 调整面板大小 | 拖动阅读器中的分隔条         |
| 复制论文 ID  | 点击阅读器工具栏中的 ID 标签 |
| 切换编辑模式 | 点击 "Vibe" / "Manual" 按钮  |
| 删除论文     | 论文页面 → 悬停 → 删除按钮   |
